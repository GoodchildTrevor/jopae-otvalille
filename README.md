# Jopae otvalille

Начало работы
-----------

1. **Создайте ТГ-бота** `@BotFather <https://t.me/BotFather>`_ и получите персональный токен.
2. **Настройте переменные окружения**:

   ```bash
    touch .env
    nano .env
    OPEN_WEATHER = ***
    AMBEE = ***

    JOPAE_BOT = ***

    TIMEZONE = ***

    LAT =  ***
    LON =  ***
    ```
