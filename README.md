Начало работы
-----------

1. **Создайте ТГ-бота** `@BotFather <https://t.me/BotFather>`_ и получите персональный токен.
2. **Set up environment variables**:

   ```bash
   cp .env.example .env
   # Edit .env:
   JOPAE_BOT=your_bot_token_here
   TIMEZONE=Europe/Moscow
    ```
